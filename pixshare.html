PixShare

L'application qui transforme le partage de photos en albums privés et sécurisés

React Native Expo Firebase Application Mobile

L'histoire derrière PixShare

Lors d'un voyage avec des amis, nous nous sommes retrouvés avec des dizaines de photos réparties sur différents téléphones. Comment rassembler toutes ces photos sans passer par des groupes WhatsApp interminables ou des drives partagés compliqués ?

Le défi : Créer un espace privé et sécurisé où chaque participant peut ajouter ses photos, accessible uniquement aux personnes ayant le code d'accès. Simple, rapide et sans inscription compliquée.

"Une app où on crée un album, on partage le code, et tout le monde peut ajouter ses photos sans s'embêter avec des comptes et des configurations compliquées !"

PixShare est né de cette simplicité recherchée : des albums photo privés accessibles par code, où chacun contribue facilement à immortaliser les moments partagés.

Fonctionnalités principales

Albums sécurisés

Chaque album est protégé par un code d'accès unique. Seules les personnes ayant le code peuvent voir et ajouter des photos.

Ajout collaboratif

Tous les membres peuvent ajouter leurs photos à l'album partagé. Plus besoin d'envoyer les photos une par une !

Codes d'accès simples

Partagez simplement le code d'accès de l'album. Pas besoin de créer des comptes ou d'ajouter des contacts.

Galerie unifiée

Toutes les photos de l'événement rassemblées au même endroit. Visualisez et téléchargez facilement toutes les photos.

Stockage cloud

Vos photos sont sauvegardées dans le cloud de manière sécurisée. Accédez-y depuis n'importe quel appareil.

Interface intuitive

Design simple et épuré pour une expérience utilisateur fluide. Créez, rejoignez et partagez en quelques clics.

Comment ça marche ?

1

Créez votre album

Donnez un nom à votre album, choisissez une photo de couverture. Un code d'accès unique est automatiquement généré.

2

Partagez le code

Envoyez le code d'accès à vos amis, famille ou collègues. Ils pourront rejoindre l'album en saisissant ce code.

3

Ajoutez vos photos

Chaque membre peut ajouter ses photos directement depuis son téléphone. Toutes les photos sont rassemblées dans l'album.

4

Profitez de vos souvenirs

Consultez, téléchargez et partagez toutes les photos de l'événement depuis un seul endroit.
