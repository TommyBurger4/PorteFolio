Guide utilisateur Créno

Application de planification d'événements en groupe

1. Onglet Agenda

L'onglet Agenda affiche votre calendrier personnel avec vos indisponibilités et événements.

Ajouter une indisponibilité

  1. Sélectionnez une date sur le calendrier
  2. Pour une plage de dates : sélectionnez une première date, puis une seconde date pour définir la fin de la période
  3. Définissez l'heure de début
  4. Définissez l'heure de fin
  5. Le bouton "Toute la journée" définit automatiquement 00:00 - 23:59
  6. Appuyez sur "Confirmer l'indisponibilité" pour enregistrer

Indisponibilités récurrentes (Premium)

  1. Sélectionnez une date unique (pas une plage)
  2. Activez l'option "Récurrence"
  3. Définissez le nombre de semaines (jusqu'à 52)
  4. L'indisponibilité sera créée chaque semaine au même jour

Supprimer une indisponibilité

  1. Faites défiler vers le bas pour voir la liste "Mes indisponibilités"
  2. Appuyez sur l'icône de suppression (corbeille)
  3. Confirmez la suppression

Conflits avec les événements

Lors de la création d'une indisponibilité qui chevauche un événement auquel vous participez :

  • Une alerte affiche la liste des événements en conflit
  • "Me désinscrire et continuer" : vous êtes retiré des événements et l'indisponibilité est créée
  • "Annuler" : votre participation aux événements est conservée
Limite gratuite : 10 indisponibilités maximum. Premium : illimitées.

2. Onglet Groupes

Les groupes permettent de planifier des événements entre plusieurs utilisateurs.

Créer un groupe

  1. Appuyez sur "Créer un groupe"
  2. Saisissez le nom du groupe (obligatoire)
  3. Saisissez une description (optionnel)
  4. Appuyez sur "Créer"
  5. Un code d'invitation à 6 caractères est généré automatiquement

Rejoindre un groupe

  1. Appuyez sur "Rejoindre un groupe"
  2. Saisissez le code d'invitation à 6 caractères
  3. Appuyez sur "Rejoindre"

Sélectionner un groupe actif

Dans la liste de vos groupes, appuyez sur "Sélectionner ce groupe". Le groupe sélectionné devient le contexte pour les événements et calendriers.

Quitter un groupe

  1. Appuyez sur "Quitter" à droite du nom du groupe
  2. Confirmez dans la boîte de dialogue

Informations affichées

  • Nom du groupe
  • Description (si renseignée)
  • Nombre de membres
  • Code d'invitation
Limite gratuite : 1 groupe maximum. Premium : illimités.

3. Création d'événements

Créer un événement

  1. Accédez à l'écran de création d'événement depuis l'onglet Groupe
  2. Saisissez le titre de l'événement (obligatoire)
  3. Saisissez une description (optionnel)
  4. Sélectionnez la date de début et la date de fin
  5. Définissez l'heure de début et l'heure de fin
  6. Sélectionnez les participants parmi les membres du groupe
  7. Les participants avec des indisponibilités en conflit sont affichés avec un indicateur orange
  8. Appuyez sur "Créer l'événement"

Indisponibilités automatiques

Lors de la création d'un événement :

  • Une indisponibilité est automatiquement créée pour chaque participant
  • Cette indisponibilité couvre la durée de l'événement
  • Elle est liée à l'événement et sera supprimée si l'événement est supprimé

Modifier un événement

  1. Appuyez sur un événement dans l'onglet "Mes Événements"
  2. Si vous êtes le créateur, vous pouvez modifier les informations
  3. Modifiez le titre, la description, les horaires ou les participants
  4. Appuyez sur "Sauvegarder les modifications"

Supprimer un événement

  1. Ouvrez l'écran de modification de l'événement
  2. Appuyez sur "Supprimer l'événement"
  3. Confirmez dans la boîte de dialogue
  4. Les indisponibilités liées sont supprimées automatiquement

4. Onglet Mes Événements

Cet écran liste tous les événements auxquels vous participez.

Indicateurs visuels

Indicateur Signification
Bordure verte Vous participez sans conflit
Bordure rouge Conflit avec une indisponibilité
Badge "TERMINÉ" L'événement est passé
Icône orange Conflit détecté

Informations affichées

  • Titre de l'événement
  • Créateur de l'événement
  • Date(s) de l'événement
  • Horaires de début et de fin
  • Liste des participants

5. Notifications

L'application envoie des notifications push pour vous tenir informé des changements.

Types de notifications

Type Description
new_event Vous avez été ajouté à un nouvel événement
participant_unavailable Un participant s'est désisté d'un événement auquel vous participez
event_deleted Un événement auquel vous participiez a été supprimé

Écran des notifications

  • Les notifications non lues sont affichées avec un indicateur bleu
  • Appuyez sur une notification pour la marquer comme lue
  • Les notifications liées à un événement redirigent vers les détails

6. Marqueurs du calendrier

Marqueur Signification
Cercle bleu (plein) Indisponibilité sur cette date
Cercle jaune (contour) Événement sur cette date
Cercle bleu avec bordure jaune Indisponibilité et événement sur la même date

Abonnement équipe

  • Un utilisateur souscrit à un abonnement équipe
  • Cet abonnement est assigné à un groupe spécifique
  • Tous les membres du groupe bénéficient des fonctionnalités Premium
  • L'abonnement peut être transféré vers un autre groupe

Gestion de l'abonnement

  • Souscrire : Sélectionnez un plan et appuyez sur "S'abonner"
  • Restaurer : Appuyez sur "Restaurer les achats" pour récupérer un abonnement existant
  • Annuler : Gérez votre abonnement dans les paramètres de l'App Store
Renouvellement automatique : Les abonnements se renouvellent automatiquement sauf annulation au moins 24h avant la fin de la période en cours.
