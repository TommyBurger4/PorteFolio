Guide d'utilisation Créno

Documentation complète de l'application Créno organisée par onglet

Onglet Agenda

L'onglet Agenda est votre espace personnel pour gérer vos disponibilités. C'est ici que vous indiquez quand vous êtes disponible ou non pour participer aux événements de vos groupes.

Ajouter une indisponibilité

1 Appuyez sur une date dans le calendrier pour la sélectionner
2 Faites défiler vers le bas (scroll down) pour voir l'interface de création d'indisponibilité
3 Sélectionnez vos heures d'indisponibilité :
  • Activez "Toute la journée" si vous êtes indisponible toute la journée
  • Ou définissez des heures spécifiques (ex: 14h00 - 18h00)
4 Appuyez sur "Confirmer l'indisponibilité" - La date apparaîtra avec un fond bleu dans le calendrier

Interactions avec le calendrier

Jour avec événement
Lorsque vous appuyez sur une date qui contient un événement (cercle jaune) et que vous faites défiler vers le bas, vous verrez le nom de l'événement.
Jour avec indisponibilité
Lorsque vous appuyez sur une date bleue (indisponibilité) et que vous faites défiler vers le bas, vous pouvez :
  • Voir vos plages d'indisponibilité pour cette date
  • Modifier ou supprimer une indisponibilité existante

Gérer toutes vos indisponibilités

Liste complète de vos indisponibilités
En descendant dans l'onglet Agenda (faites glisser vers le bas), vous accédez à la liste complète de toutes vos indisponibilités :
  • Elles sont classées par date (les plus proches en premier)
  • Chaque indisponibilité affiche la date et les horaires
  • Pour supprimer : Cliquez sur une indisponibilité pour la supprimer
  • Les indisponibilités passées sont automatiquement masquées
Important : Lorsque vous participez à un événement, des indisponibilités sont automatiquement créées pour bloquer ces créneaux. Ces indisponibilités sont liées à l'événement et seront supprimées automatiquement si l'événement est annulé.

Onglet Groupes

L'onglet Groupes affiche le calendrier de votre groupe actuel avec tous les événements et indisponibilités des membres. C'est une vue d'ensemble pour voir l'activité et la disponibilité de tout le groupe.

Calendrier du groupe
Dans cet onglet, vous voyez :
  • Tous les événements du groupe (même ceux auxquels vous ne participez pas)
  • Les cercles jaunes indiquent les jours avec des événements
  • Les cercles bleus indiquent qu'au moins un membre du groupe est indisponible ce jour
  • En appuyant sur une date avec un cercle bleu, vous voyez qui est indisponible
  • Le bouton "Changer" en haut permet de changer de groupe ou d'en gérer

Créer un groupe

1 Allez dans l'onglet "Groupes" (2ème icône)
2 Appuyez sur "Changer" en haut de l'écran
3 Appuyez sur "Gérer mes groupes"
4 Appuyez sur "Créer un groupe"
5 Donnez un nom à votre groupe
  • Exemple : "Famille", "Amis du lycée", "Équipe de foot"
  • Le nom doit être clair pour que vos amis le reconnaissent
6 Appuyez sur "Créer" - Un code unique à 6 caractères sera généré
7 Partagez le code du groupe avec vos amis pour qu'ils puissent rejoindre

Rejoindre un groupe

1 Allez dans l'onglet "Groupes"
2 Appuyez sur "Changer" en haut de l'écran
3 Appuyez sur "Gérer mes groupes"
4 Appuyez sur "Rejoindre un groupe"
5 Entrez le code du groupe (6 caractères) que votre ami vous a partagé
6 Appuyez sur "Rejoindre" - Vous êtes maintenant membre !

Onglet Événements

L'onglet Événements centralise tous les événements de vos groupes. Vous y créez de nouveaux événements et consultez ceux existants.

Créer un événement

1 Appuyez sur le bouton "+" au centre de la barre de navigation
2 Remplissez les informations de base :
  • Titre : Nom de l'événement (ex: "Dîner d'anniversaire")
  • Description : Détails supplémentaires (optionnel)
3 Choisissez le mode de recherche :
  • Heures spécifiques : Pour définir des heures de début et fin précises (ex: 14h00-18h00)
  • Durée en jours : Pour un événement sur plusieurs jours consécutifs
4 Sélectionnez les participants parmi les membres de votre groupe
5 Choisissez la date de début de recherche : À partir de quand chercher un créneau disponible
6 Appuyez sur "Rechercher des créneaux" - L'algorithme trouve automatiquement les créneaux où tous les participants sélectionnés sont disponibles
7 Choisissez un créneau parmi les propositions
8 Appuyez sur "Créer l'événement" pour confirmer

Comment fonctionne la recherche de créneaux

Algorithme intelligent
L'application analyse automatiquement :
  • Les indisponibilités de tous les participants sélectionnés
  • Les événements existants auxquels ils participent déjà
  • La durée souhaitée pour votre événement
  • Les créneaux libres sur les 3 prochains mois
Elle propose jusqu'à 10 créneaux où tout le monde est disponible.
Astuce : Si aucun créneau n'est trouvé, essayez de réduire le nombre de participants ou de modifier les heures souhaitées.

Consulter vos événements

Liste de vos événements
Dans l'onglet "Événements" (3ème icône), vous trouvez :
  • Les événements auxquels vous participez
  • Les événements que vous avez créés
  • Chaque événement affiche : titre, date, horaires et nombre de participants
Gérer vos événements créés
Pour gérer vos événements créés :
  • Appuyez sur le bouton "+" au centre de la barre de navigation
  • Descendez pour voir la section "Mes événements créés"
  • Appuyez sur un événement pour le modifier
  • Utilisez le bouton ✕ rouge pour supprimer un événement
  • La suppression notifiera tous les participants
  • Les indisponibilités liées seront automatiquement supprimées

Notifications

Types de notifications

Créno vous envoie des notifications pour :

  • Nouveaux événements dans vos groupes
  • Changements de disponibilité d'un participant
  • Événements supprimés
  • Nouveaux membres dans vos groupes

Gérer les notifications

1 Allez dans "Réglages" (dernière icône)
2 Appuyez sur "Notifications"
3 Activez ou désactivez selon vos préférences

Questions fréquentes

Je ne reçois pas de notifications

  1. Vérifiez que les notifications sont activées dans Créno (Réglages)
  2. Vérifiez les réglages iOS : Réglages > Notifications > Créno
  3. Assurez-vous d'avoir une connexion internet

Comment supprimer un groupe ?

Seul le créateur du groupe peut le supprimer. Dans l'onglet Groupes, descendez tout en bas de la page pour trouver l'option de suppression.

Puis-je modifier un événement après sa création ?

Oui ! Pour modifier ou supprimer vos événements : appuyez sur le bouton "+" (création d'événement), puis descendez dans la page pour voir la section "Mes événements créés". Vous pouvez alors appuyer sur un événement pour le modifier ou utiliser le bouton ✕ rouge pour le supprimer.

Comment quitter un groupe ?

Dans l'onglet Groupes, appuyez sur "Changer" en haut de l'écran, puis sur "Gérer mes groupes". Vous pourrez alors quitter le groupe souhaité.

Comment fonctionne le système Premium ?

La version gratuite permet :

  • 1 groupe maximum
  • 10 indisponibilités maximum

La version Premium (0,99€/mois) offre :

  • Groupes illimités
  • Indisponibilités illimitées

Les indisponibilités créées par les événements comptent-elles dans ma limite ?

Non, seules les indisponibilités que vous créez manuellement comptent dans la limite de 10 pour la version gratuite.
