Présentation de Fakt

Fakt est une application mobile professionnelle conçue pour simplifier la création et la gestion de factures pour les propriétaires de locations saisonnières. Compatible avec les réservations Booking.com, elle automatise la génération de factures conformes et leur envoi par email.

Création rapide

Générez des factures professionnelles en moins de 30 secondes

Stockage sécurisé

Toutes vos factures sont sauvegardées localement et accessibles hors ligne

Envoi automatisé

Envoyez directement les factures par email avec modèles personnalisables

PDF professionnel

Format PDF standard avec toutes les mentions légales requises

Compatibilité : L'application est disponible sur Android et iOS. Elle fonctionne également en mode hors ligne pour la création et la consultation des factures.

Créer une facture

L'onglet "Nouvelle Facture" vous permet de créer rapidement des factures professionnelles. Le formulaire est optimisé pour les réservations Booking.com avec calcul automatique des montants.

Étapes de création

1 Informations client
  • Prénom et nom du client
  • Adresse email (pour l'envoi automatique)
  • Adresse postale (optionnelle)
2 Dates du séjour
  • Date d'arrivée (format JJ/MM/AAAA)
  • Date de départ (format JJ/MM/AAAA)
  • Le nombre de nuits est calculé automatiquement
3 Montants
  • Prix par nuit (HT)
  • Taxe de séjour (montant total)
  • Le total est calculé automatiquement
4 Options Booking.com
  • Cochez "Réservation Booking" si applicable
  • Entrez le numéro de réservation Booking
  • Activez "Tarif Genius" si le client en bénéficie
5 Génération et envoi
  • Appuyez sur "Générer la facture"
  • Le PDF est créé automatiquement
  • L'email s'ouvre avec le PDF en pièce jointe
Astuce : Le numéro de facture est généré automatiquement selon le format : FACTURE-AAAAMM-XXX (où XXX est un compteur incrémental).

Paramètres

La configuration des paramètres est essentielle pour le bon fonctionnement de l'application. Ces informations apparaîtront sur toutes vos factures.

Informations du propriétaire

1 Identité
  • Prénom et nom du propriétaire
  • Email de contact
  • Numéro de téléphone
2 Établissement
  • Nom commercial (si différent)
  • Adresse complète de la location
  • Code postal et ville
3 Informations légales
  • Numéro SIRET ou identifiant fiscal
  • Numéro d'enregistrement de la location
  • Mentions légales additionnelles
Obligatoire : Ces informations sont requises pour générer des factures conformes à la législation française.

Questions fréquentes

Comment modifier une facture déjà créée ?

Les factures ne peuvent pas être modifiées après création pour des raisons légales. Vous devez supprimer la facture et en créer une nouvelle avec les bonnes informations.

Où sont stockées mes factures ?

Les factures sont stockées localement sur votre appareil dans un dossier sécurisé de l'application. Elles restent accessibles même hors ligne.

Puis-je utiliser l'application pour plusieurs locations ?

Oui, vous pouvez modifier les informations de l'établissement dans les paramètres selon la location concernée. Pensez à vérifier les informations avant chaque facture.

Comment sauvegarder mes factures ?

Nous recommandons :

  • Activez la copie cachée (BCC) pour recevoir une copie par email
  • Exportez régulièrement vos factures vers un service cloud
  • Conservez une copie sur un disque externe

L'application nécessite-t-elle une connexion internet ?

Non pour la création et consultation des factures. Oui pour l'envoi par email et les mises à jour.

Les factures sont-elles conformes légalement ?

Oui, les factures incluent toutes les mentions légales obligatoires en France :

  • Date et numéro de facture
  • Identité complète du prestataire
  • Identité du client
  • Description détaillée de la prestation
  • Montants HT, TVA et TTC
  • Mentions légales spécifiques

Comment contacter le support ?

Pour toute question ou problème technique, vous pouvez nous contacter via l'onglet "À propos" dans les paramètres de l'application.

Mise à jour : L'application est régulièrement mise à jour pour améliorer les fonctionnalités et corriger les bugs. Pensez à installer les mises à jour disponibles.

Conseils d'utilisation

Vérifiez toujours les informations avant de générer une facture. Une fois créée, elle ne peut plus être modifiée.
Créez vos factures rapidement après le départ des clients pour ne pas oublier les détails.
Organisez vos factures en utilisant un système de numérotation cohérent (automatique dans l'app).
Sauvegardez régulièrement vos factures sur un service cloud ou un disque externe.
